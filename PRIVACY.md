# Privacy Policy

**Last updated: March 2026**

## Data Storage

This application stores data **exclusively on your device**:

- **localStorage/IndexedDB**: Used to save your test progress, answers, and results
- **No data is transmitted** to any external server (except optional anonymous analytics - see below)
- All data remains under your control and can be cleared at any time by clearing your browser data

## What We Don't Collect

- Personal identification information
- Test answers or results (they stay on your device)
- Device identifiers
- Location data
- Browsing history

## Analytics (Optional)

If enabled, this application uses **privacy-friendly analytics** that collect only aggregate, non-personal data:

- Total page views
- Bounce rate
- Referrer source

**We do NOT collect:**
- IP addresses
- Cookies
- Browser fingerprints
- Any personally identifiable information

Analytics are used solely to understand if the project is being used, helping the maintainer decide whether to continue development.

## Third-Party Services

This application:
- Does not include advertising
- Does not include social media trackers
- Does not sell or share any data
- Only loads analytics script if analytics are enabled

## Your Control

You can clear all locally stored data at any time by:
1. Clearing your browser's site data for this application
2. Uninstalling the PWA
3. Disabling cookies for this site
