# Contributing

## We Are Not Accepting Contributions

This is a personal hobby project, and we are not currently accepting:

- Pull requests
- Feature requests
- Translation contributions
- Question submissions

## Why?

This project was created as a personal study tool. The scope and direction are intentionally limited to specific features as determined by the maintainer.

## How to Report Issues?

While we are not accepting contributions, if you find a factual error in the question bank (e.g., an incorrect answer or citation), you may open an issue on GitHub to report it. The maintainer will review reported errors but cannot guarantee fixes.

## Alternatives

If you need a similar tool with different features or community contributions, consider:
- Forking the repository (under MIT license)
- Using one of the many citizenship test apps available in app stores
